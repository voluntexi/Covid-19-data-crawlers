# Covid-19-data-crawlers
爬取新冠肺炎数据 做成可视化图表

# 所需的库有
* json：解析成字符串/字典
* requests:爬取
* re：正则表达式获取时间
* pycharts：可视化图表
* lxml：可以迅速、灵活地处理 XML 和 HTML
* openpyxl：excel写入
